# Claude Agent Manager

Gestor de agentes especializados para Claude Code.

## Instalación

```bash
git clone https://github.com/GailenTech/claude-agent-manager.git
cd claude-agent-manager
./install.sh
```

## Uso

```bash
agent-manager
```

### Controles
- `↑/↓` Navegar
- `SPACE` Seleccionar
- `1` Vista Usuario
- `2` Vista Proyecto
- `s` Guardar
- `v` Ver agente
- `q` Salir

## Desinstalar

```bash
./uninstall.sh
```
